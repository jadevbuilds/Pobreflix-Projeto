# 🎬 POBREFLIX - Streaming Experience

O **Pobreflix** é uma plataforma de catálogo de filmes e séries desenvolvida com tecnologias de ponta para oferecer uma interface fluida, moderna e totalmente responsiva. O projeto foca na experiência do usuário (UX), trazendo um design inspirado nos grandes players de streaming.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando o que há de mais moderno no ecossistema Web:

* **Next.js 14** (App Router) - Para uma navegação rápida e otimizada.
* **React** - Para construção de interfaces modulares.
* **Tailwind CSS** - Para estilização personalizada e design responsivo.
* **Lucide React / Heroicons** - Para ícones modernos.
* **TypeScript** - Para garantir a segurança e organização do código.

## ✨ Funcionalidades

* **Interface Premium:** Design escuro com grid de 8 colunas para máxima densidade de conteúdo.
* **Busca em Tempo Real:** Sistema de filtro inteligente que varre o catálogo de filmes e séries instantaneamente.
* **Sistema de Abas:** Navegação entre Lançamentos, Recentes, Mais Vistos e Em Alta (🔥).
* **Totalmente Responsivo:** Adaptado para celulares, tablets e computadores.
* **Categorização:** Seções separadas para Filmes Online e Séries Online.
